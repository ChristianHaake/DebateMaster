# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-21

### Added
- Standard haak3 UI shell (Header, Footer, Design Tokens)
- Zustand store for state management
- Participant Manager component (add/remove participants)
- Debate Controller with independent timers for each speaker
- Analytics component with a bar chart for speaking times
- JSON Export for debate data
- Complete Vite + React + TypeScript setup following the haak3 standard

### Changed
- Migrated initial project plan into a full development setup
- Replaced standard placeholders in documentation and legal pages
